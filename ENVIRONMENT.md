# Environment

This project uses `uv` with Python 3.11.9. The intended and sole compute backend for the confirmatory study is Apple Silicon / MPS. All reported results are to be produced on MPS; no CUDA alternative is part of this pre-registration. The reproducibility claim is bitwise-within-MPS only, with seeds fixed and recorded for each run.

Hardware: MacBook Pro, Apple M2 Max (8P + 4E), 96 GB. OS: macOS 15.7.4 (Darwin 24.6.0).
