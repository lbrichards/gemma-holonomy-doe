# log sin φ check

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A belt-and-suspenders balance diagnostic requiring |mean(log sin φ_real) - mean(log sin φ_shuffled)| < 0.045.

## Why this term exists

The 0.045 threshold is ~1/5 of the materiality threshold (0.223), ensuring φ imbalance is negligible relative to the effect size of interest.
