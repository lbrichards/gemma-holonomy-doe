# shuffled-feature plane

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A control plane pairing one active SAE feature with one inactive dictionary feature, magnitude-matched to the real-feature plane.

## Why this term exists

Disrupts semantic coherence while preserving feature-direction structure. The real-vs-shuffled contrast is the powered H-sem test, isolating meaning from magnitude and feature geometry.
