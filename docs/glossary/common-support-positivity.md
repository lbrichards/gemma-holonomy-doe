# common support / positivity

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The requirement that all treatment groups have overlapping covariate distributions; the conceptual basis for the common-support band ensuring magnitude matching does not extrapolate.

## Source

Causal inference (Rosenbaum & Rubin); propensity-score methods
