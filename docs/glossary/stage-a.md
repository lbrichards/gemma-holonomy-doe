# Stage A

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The blind setup phase that extracts activations, selects planes, computes the magnitude band, constructs centers, measures covariates, and produces the manifest.

## Why this term exists

Stage A is structurally incapable of computing or storing response values. This enforces the pre-registration principle that all design decisions are frozen before holonomy is observed.
