# NULL-ATTRIBUTED verdict

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A special FALSIFIED branch for H-sem when an unadjusted effect vanishes under covariate adjustment, attributing the apparent effect to confounds.

## Why this term exists

If real-vs-shuffled differs unadjusted but not adjusted, the signal came from manifold distance or phi, not from meaning. This is a publishable null, not a failure to detect.
