# pre-registration

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The practice of publicly committing to hypotheses, methods, and analysis plans before observing data, preventing post-hoc hypothesizing.
