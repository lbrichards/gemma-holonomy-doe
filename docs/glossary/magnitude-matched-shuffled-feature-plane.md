# magnitude-matched shuffled-feature plane

**`coined`**

A control plane constructed by pairing one active SAE feature with one inactive real dictionary feature, matched to real-feature planes in pullback-metric in-plane magnitude.

## Why this term exists

Provides a semantically degraded control that preserves magnitude structure, isolating the contribution of semantic coherence to holonomy from the confounding effect of activation magnitude.
