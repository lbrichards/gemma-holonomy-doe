# holonomy-magnitude response

**`coined`**

The measured relationship between in-plane magnitude and holonomy (rotation per enclosed area), the primary response variable testing H-mag.

## Why this term exists

Quantifies how holonomy scales with magnitude to disentangle magnitude effects from semantic structure effects, enabling the factorial design to isolate each factor's contribution.
