# covariate adjustment

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

Regression adjustment for manifold distance and phi in the real-vs-shuffled contrast, applied by default regardless of balance.

## Why this term exists

Even if balance diagnostics look good, adjustment removes any residual confounding. The adjusted contrast is the primary H-sem test; unadjusted is reported alongside for comparison.
