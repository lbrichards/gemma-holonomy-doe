# determinant

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

A scalar value encoding the signed volume scaling factor of a linear transformation, with det = 0 indicating singularity.

$$
\det(M)
$$

Linear algebra
