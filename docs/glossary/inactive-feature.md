# inactive feature

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

An SAE feature with zero or negative activation (code <= 0) at a given base point, used as the second direction in shuffled-feature planes.

## Why this term exists

Pairing an active feature with an inactive one breaks the joint activation structure while still using a real dictionary direction, isolating semantic coherence from feature geometry.
