# survivor

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A base point that passes all filters (64-token minimum, det M floor, valid plane selection) and enters the experiment sample or reserve.

## Why this term exists

Not all drawn articles yield usable base points — some are too short, some have degenerate planes. "Survivor" distinguishes points that make it through the full selection pipeline.
