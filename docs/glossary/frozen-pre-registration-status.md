# frozen (pre-registration status)

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The status indicating that the pre-registration document is locked and no substantive changes may be made, only blind clarifications via addenda.

## Why this term exists

FROZEN status is the guarantee that hypotheses, methods, and analysis plans were committed before any holonomy was observed, preventing post-hoc hypothesis modification.
