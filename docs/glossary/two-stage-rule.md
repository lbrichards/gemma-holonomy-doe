# two-stage rule

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The pre-specified procedure where a second batch of N₂ = 96 base points is run only if stage-1 H-sem is INCONCLUSIVE, with both results reported.

## Why this term exists

If H-sem lands inconclusive after stage 1, the true effect may be smaller than planned for. The second stage is pre-declared with fixed trigger and size, avoiding optional-stopping bias.
