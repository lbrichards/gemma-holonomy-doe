# experiment sample

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The first 390 surviving base points (in seed order, excluding burned pilot points), constituting the fixed-N dataset for confirmatory analysis.

## Why this term exists

The sample size is pre-registered and powered at 0.90 for the material effect. All 390 points are run to completion and analyzed once, with no peeking or result-dependent stopping.
