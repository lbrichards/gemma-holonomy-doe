# materiality threshold

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The minimum effect size log(1.25) = 0.2231 on the log scale (a 25% multiplicative change) below which effects are not considered material.

$$
\delta = \log(1.25) = 0.2231
$$

## Why this term exists

Defines "material" for all three hypotheses. An effect below this threshold is too small to matter scientifically, even if statistically significant. Powers the 0.90 power calculation.
