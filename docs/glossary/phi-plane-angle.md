# phi (plane angle)

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The mutual Euclidean angle between the two raw normalized plane directions, computed as arccos(d1 · d2).

$$
\phi = \arccos(d_1 \cdot d_2)
$$

## Why this term exists

Although the primary response uses pullback-metric area, phi captures native Euclidean feature geometry, which is semantically loaded and must be balance-checked.
