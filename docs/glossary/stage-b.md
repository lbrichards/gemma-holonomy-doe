# Stage B

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The measurement phase that reads a Stage A manifest and writes holonomy response values to a separate artifact, never mutating the manifest.

## Why this term exists

Separation of setup (Stage A) and measurement (Stage B) ensures the blind boundary is maintained. Stage B cannot influence plane selection, band decisions, or covariate measurement.
