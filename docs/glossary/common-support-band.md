# common-support band

**`coined`**

The magnitude range where all three plane-type arms (random, shuffled-feature, real-feature) have overlapping in-plane magnitude distributions, enabling matched comparisons.

## Why this term exists

The three plane arms have widely separated magnitude distributions (real p50 ~65, shuffled ~59, random ~13). Matching is only meaningful where distributions overlap; cutting levels on one arm alone would force extrapolation rather than a true match.
