# three-arm band

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The overlap of the [p5, p95] magnitude ranges across all three plane arms (real, shuffled, random), required to have width >= 0.5 × pooled IQR.

## Why this term exists

Defines the primary common-support region where magnitude-matched comparisons are valid across all three arms, enabling the full H-grad monotone gradient test.
