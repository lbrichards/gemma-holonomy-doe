# log-scale paired contrast

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The within-base-point difference d_b = log H_real - log H_shuffled, the primary test statistic for H-sem.

$$
d_b = \log H_{\text{real}} - \log H_{\text{shuffled}}
$$

## Why this term exists

Taking differences within each base point removes base-point-level variation, maximizing power. The log scale makes the materiality threshold (0.2231) a multiplicative 25% change.
