# factorial design

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

An experimental design that crosses multiple factors (here plane type × magnitude level), enabling estimation of main effects and interactions.

Fisher (1935); experimental design
