# factorial design

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

An experimental design that crosses multiple factors; here 3 plane types × 2 magnitude levels enables separating H-sem (semantic effect) from H-mag (magnitude effect).

## Source

Fisher (1935); experimental design
