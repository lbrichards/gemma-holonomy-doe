# real-feature plane

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A plane spanned by two jointly active SAE features at the base point, representing intact semantic structure.

## Why this term exists

Jointly active features capture the model's learned semantic geometry at the base point. This arm is the upper anchor of the semantic gradient tested by H-grad.
