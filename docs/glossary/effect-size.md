# effect size

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

A standardized measure of the magnitude of a phenomenon, independent of sample size, used for power analysis and scientific interpretation.

## Source

Cohen (1988); meta-analysis literature
