# H-mag

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The magnitude hypothesis predicting that holonomy increases with in-plane pullback-metric magnitude, pooled across plane types.

## Why this term exists

Tests whether activation magnitude is a main effect on holonomy, independent of semantic structure. A positive H-mag would indicate curvature scales with how far from the base point we probe.
