# active feature

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

An SAE feature with positive activation (code > 0) at a given base point, indicating the feature fires on that input.

## Why this term exists

Active features define the semantic content the SAE attributes to the base point. The real-feature arm uses pairs of jointly active features to preserve this semantic structure.
