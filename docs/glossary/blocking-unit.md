# blocking unit

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The unit (here base point) within which all treatment combinations are observed, reducing between-unit variance.

## Source

Experimental design (randomized complete block)
