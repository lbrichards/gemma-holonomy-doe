# stage-1 / stage-2 partition

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The pre-registered split of the draw pool into 96 stage-1 points, 96 stage-2 reserve, and 48 unused reserve, determined by draw order before any holonomy.

## Why this term exists

Fixed stage boundaries allow the two-stage rule to be fully pre-specified — which points constitute stage 1 vs stage 2 is determined by seed-42 draw order, not by any response values.
