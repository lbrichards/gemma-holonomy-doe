# residual stream

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The additive information flow through a transformer where each layer adds its contribution to the accumulated hidden state; the resid_post site is where SAEs are trained and base points extracted.
