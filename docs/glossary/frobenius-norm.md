# Frobenius norm

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The square root of the sum of squared matrix elements; used to compute the symmetric residual diagnostic for non-rotational distortion in the transport operator.

$$
\|A\|_F = \sqrt{\sum_{ij} A_{ij}^2}
$$

## Source

Linear algebra
