# reserve pool

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

Base points beyond the 390 experiment sample, held back for dropout replacement or future analysis but never analyzed as part of the confirmatory study.

## Why this term exists

Provides a safety margin if some experiment-sample points fail extraction or plane selection, without introducing selection bias from result-dependent sampling.
