# pairing rule

**`coined`**

The fixed procedure for constructing plane arms before holonomy computation, specifying how SAE features are selected and combined for each plane type.

## Why this term exists

Ensures reproducible and pre-registered plane construction across the three arms: real-feature uses jointly active features, shuffled-feature uses one active plus one inactive, random uses two random unit directions.
