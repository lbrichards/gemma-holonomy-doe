# draw order

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A sequential integer assigned to each surviving WikiText article after seed-42 shuffling, determining the experiment-vs-reserve partition.

## Why this term exists

The draw order provides a reproducible, seed-determined sequence that defines which points enter the experiment sample (first 390) versus reserve, independent of any response values.
