# two-arm terminal collapse

**`coined`**

The fallback case where even the two-arm real+shuffled common-support band fails the minimum width requirement, making matched semantic comparison impossible.

## Why this term exists

Added as a pre-data clarification to handle the edge case where real and shuffled features also do not share a common magnitude regime. H-sem is then reported as undefined at matched magnitude.
