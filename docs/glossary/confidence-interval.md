# confidence interval

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

An interval estimate of a parameter such that repeated sampling would capture the true value a specified proportion (e.g., 95%) of the time; compared against the materiality threshold to assign verdicts.

## Source

Frequentist statistics (Neyman)
