# fixed-N design

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

An experimental design where sample size is determined in advance and all observations are collected before analysis, with no peeking or early stopping.

## Why this term exists

The N = 390 (or 96 per stage) is fixed before data collection. This prevents optional-stopping problems that inflate false positives, making the null verdict interpretable as material absence.
