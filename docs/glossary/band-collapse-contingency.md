# band-collapse contingency

**`coined`**

The pre-registered fallback procedure when the three-arm common-support band is empty or too narrow to support matched magnitude comparisons.

## Why this term exists

If the arms do not share a common magnitude regime, this is itself a reportable finding. The contingency defines how to proceed with two-arm analysis for H-sem and marks H-grad as undefined.
