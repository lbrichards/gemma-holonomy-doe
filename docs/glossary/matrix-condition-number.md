# matrix condition number

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The ratio of the largest to smallest singular value; monitored at each transport step to flag numerical instability in the restricted Jacobian.

$$
\kappa(A) = \frac{\sigma_{\max}}{\sigma_{\min}}
$$

## Source

Numerical linear algebra
