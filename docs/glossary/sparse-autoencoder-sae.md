# sparse autoencoder (SAE)

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

A neural network that learns a sparse overcomplete dictionary for reconstructing activations, decomposing them into interpretable feature directions; provides the feature-based planes that define the semantic structure hypothesis.

## Source

Olshausen & Field (1996); Anthropic (2023); Gemma Scope (Google DeepMind)
