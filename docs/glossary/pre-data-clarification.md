# pre-data clarification

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A post-freeze addition to the pre-registration that is provably outcome-independent because it is recorded before any response values are computed.

## Why this term exists

Some operational details (pairing rule, article reconstruction, terminal collapse case) were refined after freeze but before any holonomy was observed. Timestamped addenda preserve integrity.
