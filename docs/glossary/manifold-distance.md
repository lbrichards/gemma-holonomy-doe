# manifold distance

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

A nuisance covariate measuring how far loop points lie from the learned manifold, via SAE reconstruction error and/or reference distribution distance.

## Why this term exists

If real-feature loops stay closer to the manifold than shuffled-feature loops, any holonomy difference could be confounded by proximity to learned structure rather than semantic coherence.
