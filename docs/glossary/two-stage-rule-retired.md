# two-stage rule (retired)

<span style="background-color: #f44336; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">⛔ RETIRED</span>

> **This term is retired.** It is not part of the current v2 design and has no replacement.

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

[RETIRED in v2] The v1 rescue mechanism where a second batch of N₂ = 96 would run if stage-1 H-sem was INCONCLUSIVE; removed when N was re-derived to 390.

## Why this term exists

v1 was powered at N = 96 with τ = 0.649 (resid_pre). After the hook-site correction, τ rose to 1.30 and N was re-derived to 390 for adequate power. At this sample size, the two-stage rescue became unnecessary and was retired in v2 §6.3.
