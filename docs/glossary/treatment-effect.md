# treatment effect

<span style="background-color: #448aff; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">INHERITED</span>

The causal difference in outcomes attributable to a treatment, here the holonomy difference between plane types.

## Source

Causal inference (Rubin causal model)
