# stage-1 / stage-2 partition (retired)

<span style="background-color: #f44336; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">⛔ RETIRED</span>

> **This term is retired.** It is not part of the current v2 design and has no replacement.

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

[RETIRED in v2] The v1 split of the draw pool into 96 stage-1 + 96 stage-2 points; superseded by the single-stage N = 390 design.

## Why this term exists

This partition supported the v1 two-stage rule at N = 96. When the hook-site correction required re-deriving N to 390, the two-stage mechanism was retired and the partition became moot. v2 uses a single-stage design with 390 experiment-sample points.
