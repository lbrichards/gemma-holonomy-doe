# pooled IQR

<span style="background-color: #4051b5; color: white; padding: 0.2em 0.6em; border-radius: 0.25em; font-size: 0.85em; font-weight: 600;">COINED</span>

The interquartile range computed over the concatenated magnitude values from all arms in a band computation, used to set the minimum width threshold.

## Why this term exists

The band width threshold is defined as 0.5 × pooled IQR to ensure the common-support region is substantive relative to the overall magnitude spread, not just technically non-empty.
